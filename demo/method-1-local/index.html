ADK Client - Method 1: Local Installation Demo

About This Method

This demo shows how to use the ADK Client Web Component after installing it via npm. The component is loaded from local node_modules using relative paths.

Key Feature: The component uses Shadow DOM with embedded styles - no external CSS needed!

Setup: Run npm install adk-client-web-component in this directory.

Widget Mode Demo

The chat widget appears in the bottom-right corner of this page. Click on it to open the chat interface.

<!-- Web component with configuration attributes -->
<adk-client
  api-url="http://localhost:8000"
  app-name="my_sample_agent"
  user-id="user_123"
  session-id="session_123"
  mode="widget"
  response-mode="stream">
</adk-client>

<!-- Import the standalone component -->
<script type="module" src="./node_modules/adk-client-web-component/dist/adk-client-standalone.js"></script>
Note: No external CSS required! The component uses Shadow DOM with embedded styles. Just serve the HTML through a web server (e.g., npx serve, python -m http.server).
