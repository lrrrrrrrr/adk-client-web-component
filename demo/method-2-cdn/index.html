About This Method

This demo shows how to use the ADK Client Web Component directly from CDN without any npm installation. The component is loaded from unpkg.com using full URLs.

Advantages: No installation required, works with any static hosting, easy to integrate into existing websites.

Widget Mode Demo

The chat widget appears in the bottom-right corner of this page. Click on it to open the chat interface.

<!-- Include the CSS from CDN -->
<link rel="stylesheet" href="https://unpkg.com/adk-client-web-component/dist/adk-client-web-component.css">

<!-- Web component with configuration attributes -->
<adk-client
  api-url="http://localhost:8000"
  app-name="my_sample_agent"
  mode="widget">
</adk-client>

<!-- Import as ES module from CDN -->
<script type="module">
  import 'https://unpkg.com/adk-client-web-component/dist/adk-client-standalone.js';
</script>
✓ No installation required! This works with any static hosting provider.

Production Configuration

Example of how you might configure this for a production environment:

<adk-client
  api-url="https://api.mycompany.com"
  app-name="customer_support"
  user-id="website_visitor">
  mode="widget"
  response-mode="stream">
</adk-client>
Security Note: In production, ensure your ADK server has proper CORS configuration and only allows requests from your domain.

Version Pinning (Recommended for Production)

For production use, it's recommended to pin to a specific version:

<!-- Pin to specific version -->
<link rel="stylesheet" href="https://unpkg.com/adk-client-web-component@0.0.3/dist/adk-client-web-component.css">
<script type="module">
  import 'https://unpkg.com/adk-client-web-component@0.0.3/dist/adk-client-standalone.js';
</script>
