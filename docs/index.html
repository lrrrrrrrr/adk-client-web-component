What is it?

A modern, self-contained web component that provides a beautiful chat interface for Google's Agent Development Kit (ADK). Built with React 19, TypeScript, and Tailwind CSS, packaged as a framework-agnostic web component with Shadow DOM and embedded styles.

Quick Start

Just add this to your HTML - that's it! No installation, no CSS files, no build step.

<!DOCTYPE html> <html> <body> <!-- Add the web component --> <adk-client api-url="http://localhost:8000" app-name="my_sample_agent" mode="widget"> </adk-client> <!-- Import from CDN --> <script type="module"> import 'https://unpkg.com/adk-client-web-component@0.0.4/dist/adk-client-standalone.js'; </script> </body> </html>
